# Firebase Configuration Setup

⚠️ **IMPORTANT:** Never commit `google-services.json` or `firebaseConfig.js` to version control!

## Setup Instructions:

1. Download `google-services.json` from Firebase Console
2. Place it in the project root directory
3. Copy `firebaseConfig.example.js` to `firebaseConfig.js`
4. Update the values in `firebaseConfig.js` with your actual Firebase credentials

These files are in `.gitignore` and will not be committed to git.
